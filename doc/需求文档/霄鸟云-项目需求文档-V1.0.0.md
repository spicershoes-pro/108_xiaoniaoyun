## 1. 产品概述

**产品名称**：霄鸟云（跨境玩具选品平台）

**核心定义**：面向跨境玩具供应链的 B2B 选品协作平台，连接全球买家与中国工厂。

**一句话定位**：以询盘 → 报价 → 采样 → 下单为核心闭环，提供从选品发现到订单交付的完整链路。

### 1.1 核心价值

| 维度 | 价值 |
|------|------|
| 买家 | 一站式发现合规货源，降低搜寻成本，保障 IP 合规 |
| 工厂 | 获取海外采购意向，展示资质，管理全流程订单 |
| 平台 | 撮合交易抽佣（5%）、IP 授权通道、供应链数据沉淀 |

### 1.2 服务场景

产品选品、工厂发现、询盘报价、样品申请、订单交付、IP 授权、玩具圈社区、汇率/热销榜工具、商家供应链与提现结算。

### 1.3 商业边界

**平台承接**：信息撮合、IP 授权通道、佣金结算（5%）、内容与商家资质审核。

**平台不承接**：见 §0.4。

## 3. 角色与权限（摘要）

| 等级 | 代码 | 角色 | 说明 |
|------|------|------|------|
| L0 | guest | 游客 | 只读公开内容 |
| L1 | buyer | 买家 | 询盘、订单、收藏、消息等 |
| L2 | merchant | 商家 | 产品、询盘、订单、样品、提现 |
| L3 | support | 客服 | 查看，不可改系统配置 |
| L4 | admin | 运营管理员 | 审核、用户/商家/内容管理 |

**关键规则**：财务结算页、系统设置页仅 L5；管理端最低 L4；商家端仅 merchant 角色可登录。

## 4. 核心业务流程

### 4.1 交易主链路

```
买家浏览/搜索 → 发起询盘 → 商家报价 → 买家下单
    → 付款（定金，默认 50%）→ 备料 → 生产 → 发货
    → 确认收货 → 完成（平台计佣 5%）→ 商家提现审批
```

### 4.2 订单状态机（9 态）

`pending_payment` → `paid` → `material` → `production` → `shipping` → `completed`

分支：`cancelled`（待付/已付可取消）、`dispute`（运输/已送达可发起，管理员调解）

### 4.3 询盘状态机

`pending` → `quoted` / `negotiating` → `converted`（下单）或 `closed`

### 4.4 其他子流程

- **样品**：pending → processing → shipped / rejected
- **IP 授权**：浏览库 → 申请 → 平台批准/拒绝
- **消息**：创建会话 → 3 秒轮询增量拉取
- **产品上架**：商家提交 pending → 平台审核 → online
