## 1. 设计规范

| 规则 | 说明 |
|------|------|
| 数据库名 | `xiaoniao` |
| 字符集 | utf8mb4 / utf8mb4_unicode_ci |
| 主键 | BIGINT UNSIGNED AUTO_INCREMENT |
| 时间 | `created_at`、`updated_at`（DATETIME）；业务时间 `*_at` |
| 金额 | DECIMAL(10,2) 单价；DECIMAL(12,2) 订单/提现；DECIMAL(15,2) GMV |
| 布尔 | TINYINT(1)，0/1 |
| 命名 | 表名、字段 snake_case |
| 外键 | schema 中部分表建 FK；应用层须保证引用一致 |
| 操作日志 | `operation_logs` 只追加，不可删改 |

## 3. 核心表字段摘要

### 3.1 users

| 字段 | 说明 |
|------|------|
| phone | 手机号，唯一登录凭证 |
| role | buyer / merchant / admin / super_admin |
| status | pending / active / suspended |

### 3.2 buyer_profiles

| 字段 | 说明 |
|------|------|
| level | bronze / silver / gold / platinum |
| total_gmv | 累计 GMV |
| credit_score | 信用分 0～100 |
| verified | 实名认证 |

### 3.3 merchant_profiles

| 字段 | 说明 |
|------|------|
| status | reviewing / active / suspended / rejected |
| rating / total_orders / total_gmv | 商家经营指标 |
| bank_* | 收款信息 |

### 3.4 products

| 字段 | 说明 |
|------|------|
| sku | 唯一，格式 XN-{序号}-{随机码} |
| status | draft / pending / online / offline / rejected |
| base_price / moq / stock | 价格、起订量、库存 |
| sales_count / view_count | 销量、浏览量 |

### 3.5 inquiries

| 字段 | 说明 |
|------|------|
| status | pending / quoted / negotiating / converted / closed |
| priority | low / medium / high |
| quote_price / quote_note / quoted_at | 商家报价 |
| message | 询盘内容（≥5 字） |

### 3.6 orders

| 字段 | 说明 |
|------|------|
| order_no | XN+YYYYMMDD+4 位序号 |
| status | 9 种订单状态 |
| total_amount / deposit / platform_fee | 金额与佣金 |
| express_company / express_no | 物流信息 |
| inquiry_id | 来源询盘（可选） |

### 3.7 messages

| 字段 | 说明 |
|------|------|
| type | text / image / file / product_card / system |
| content | 消息正文 |
| metadata | JSON，产品卡片等 |

### 3.8 system_configs（键值）

| key | 默认值 | 说明 |
|-----|--------|------|
| platform_name | 霄鸟云 | 平台名称 |
| platform_fee_rate | 0.05 | 佣金率 |
| min_withdrawal | 1000 | 最低提现（元） |
| inquiry_expire_days | 30 | 询盘有效期 |
| sms_code_expire_minutes | 5 | 验证码有效期 |

## 5. 表关联（逻辑）

```
users ──1:1── buyer_profiles
users ──1:1── merchant_profiles
merchant_profiles ──1:N── products / merchant_certs / merchant_categories
products ──1:N── product_price_tiers / product_images / product_certs
inquiries ──1:N── inquiry_items ──→ products
inquiries ──0:1── orders
orders ──1:N── order_items / order_status_logs
orders ──0:1── order_disputes
conversations ──1:N── conversation_participants / messages
ip_licenses ──1:N── ip_applications
users ──1:N── favorites / cart_items / notifications
```

## 6. 唯一性与索引要点

| 约束 | 说明 |
|------|------|
| users.phone | 全局唯一 |
| products.sku | 全局唯一 |
| orders.order_no | 全局唯一 |
| favorites (user_id, product_id) | 唯一 |
| cart_items (user_id, product_id) | upsert |
| conversation_participants (conversation_id, user_id) | 唯一 |
| product_certs (product_id, name) | 唯一 |
| 同两人仅一会话 | 应用层校验 |

高频索引：`merchant_id`、`status`、`buyer_id`、`created_at`；产品名 FULLTEXT。
