## 1. 项目目标

交付霄鸟云跨境玩具 B2B 选品平台，实现：

1. 买家选品、询盘、下单、样品、消息全链路  
2. 商家产品、询盘报价、订单履约、提现申请  
3. 平台审核（商家/产品/内容/IP）、订单监控、纠纷调解、财务结算  
4. IP 授权、玩具圈、汇率与热销榜等配套能力  
5. 三端 Vue + PHP API 可本地运行、可 Docker/生产部署

## 4. 三端开发顺序

```
后端 API 骨架与认证（S1～S2）
    → 用户端核心链路：选品→询盘→订单（S3）
    → 商家端：询盘报价→订单发货（S4）
    → 管理端：审核与监控（S5）
    → 消息/样品/IP/社区等（S3～S5 并行）
    → 部署与全链路验收（S6～S7）
```

## 6. Definition of Done

1. 实现与【201-01】字段、状态、错误码、文案一致  
2. 接口与【201-03】路由、请求/响应体一致  
3. 三端页面与【201-05】页面数、弹窗、权限显隐一致  
4. 主链路：询盘 → 报价 → 下单 → 付款 → 发货 → 完成可走通  
5. 【201-05】P0 验收项全部通过  
6. 边界内「不做」项未实现（在线支付、自建物流等）
